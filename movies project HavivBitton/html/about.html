    • About This Project

    This is a website building project, designed by inspiration of 'APPLE Music', to showcase the ability to fetch and display movie data from The Movie Database (TMDb) API.

    Features

    • Home Page with 'Popular','Top-Rate' and 'Now-Playing' movies lists!
    • Search functionality for finding movies.
    • 'Movie-Page' with details ,cast and similar movie list!
    • Responsive design for various screen sizes.

    Technologies Used

    The project is built using:

    • HTML
    • CSS
    • JavaScript (ES6)
    • TMDb API

    Future Enhancements

    In future iterations, features such as user authentication, movie reviews, and a 'Light-Mode' will be added.
